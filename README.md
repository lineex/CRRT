# CRRT Precision Management Tool (v2.6)

CRRT (Continuous Renal Replacement Therapy) Precision Management Tool 是一款专为重症医学科 (ICU) 血液净化治疗量身定制的高级决策辅助单页面网页应用 (Single Page Application)。它通过集成多组分智能换算、流速整合计算以及体外抗凝模型模拟，极大简化了复杂的 CRRT 临床液体制备与参数调节流程。

# 网址：https://nanshancrrt.netlify.app
---

## 核心功能模块

### 1. Formula Calculator (配液与化验配方计算器)
**应用场景**：通过纯粹的成分组合，计算并展示自配液（或成品液+添加剂）的最终离子浓度。支持 **“调整模式”**（基于现有成品液或处方微调）与 **“生成模式”**（从零开始计算不同添加剂混合后的浓度）。

### 2. Parameter Scheme (CRRT 参数与流速方案生成)
**应用场景**：整合了置换液种类、各通路流速（前/后置换、透析液）、以及抗凝剂（枸橼酸等）设置，输出最核心的临床指导参数：**CRRT 总剂量** 及 **进入患者体内的混合后实际离子浓度**，同时内嵌枸橼酸抗凝与游离钙估算的高阶模型。

### 3. Volume Management Wizard (容量管理决策向导)
**应用场景**：基于标准的血流动力学评估步骤，通过四步交互问卷为 ICU 医师提供精准的容量干预建议（如液体复苏优先、启用血管活性药物、或启动脱水等）。

## 核心计算模型与原理

### A. 物质的量换算系数原理 (Molar Mass Conversions)
工具底层使用标准摩尔质量将临床常用浓度（如 15% KCl）转换为毫摩尔 (mmol)，用于精确叠加计算：
- **15% KCl (氯化钾)**：13.41 mmol/g
- **10% NaCl (氯化钠)**：17.11 mmol/g
- **5% NaHCO₃ (碳酸氢钠)**：11.90 mmol/g
- **25% MgSO₄·7H₂O (硫酸镁)**：4.06 mmol/g
- **10% 葡萄糖酸钙**: 提供 223.0 mmol/L 的纯钙离子

*计算公式例如：* `K+ (mmol) = 15% KCl添加量(mL) × 0.15 ÷ 74.55 × 1000`

### B. 溶液离子浓度整合推导 (Tab 1 & Tab 2 基础推导)
系统通过分离并累计所有液体分量（基础液及各类添加剂的体积占比积）计算出混合后单位体积的各离子浓度。
- **总注入体积 (mL)** = 基础液由于添加剂增加后的总物理体积
- **特定离子浓度 (mmol/L)** = 总离子物质的量(mmol) ÷ (基础液体积(L) + 添加剂体积(L))

*注意：*新版本中严格遵循了无机盐配方的水合分子量规范修正 (例如无水葡萄糖与一水葡萄糖的影响)。

### C. CRRT 总剂量与流速整合计算 (CRRT Total Dose)
**计算方法：**
CRRT 剂量的核心是指机体与体外循环发生的液体交换率，等于所有置换与透析液流速之和加上纯脱水量。
- **置换与透析流速之和** = CVVH 前置换流速 + CVVH 后置换流速 + CVVHDF 透析液流速
- **总注入流速 (Total Infused Rate)** = 置换与透析流速之和 + 10%糖酸钙流速 + 5%碳酸氢钠流速 + **抗凝剂流速(枸橼酸)**
- **CRRT 总剂量 (ml/h)** = **总注入流速 + 净脱水量 (UF)**

*原理*：将“抗凝剂速度”等辅助液体视为置换液的一部分，进一步提高了剂量的精确度。

### D. 枸橼酸抗凝模型及流速计算 (Citrate Anticoagulation)
当选择枸橼酸抗凝（3% 或 4%）时，系统将基于 **目标抗凝浓度** (Target Citrate Concentration) 和 **血流速度 (Qb)** 计算枸橼酸泵入速度。
**计算方法：**
- `枸橼酸泵速 (ml/h) = (目标抗凝浓度(mmol/L) × 血流速度(ml/min) × 60) ÷ 对应浓度枸橼酸溶液的当量(mmol/L)`
*(例如 4% 枸橼酸钠所含枸橼酸根约为 136 mmol/L)*

**代谢后预测碱剩余/总 HCO₃⁻：**
枸橼酸在肝脏、肌肉组织代谢为碳酸氢根 ($1 \text{ mmol Citrate} \rightarrow 3 \text{ mmol HCO}_3^-$)。考虑滤器对枸橼酸的清除丢失（可配置，如 22.5%），系统能预测这部分枸橼酸进入人体后产生的碱负荷：
- `代谢后预测 HCO3- = 原液中HCO3-浓度 + (枸橼酸浓度(基于总注入流速) × (1 - 枸橼酸清除率) × 3)`

### E. 滤器内游离钙的精准预估 (Ionic Calcium in Filter)
为预警枸橼酸抗凝过程中滤器内是否会发生凝血，系统提供极其精准的 **“Ca²⁺ (估计滤器钙)”** 估算。

**原理背景：**
钙离子的稀释和螯合主要发生在**滤器前**。进入滤器的总液体流速包括：患者自身血流 (Qb)、**前置换流速 (Pre-dilution)** 以及枸橼酸抗凝剂泵注流速。*(后置换流速因在滤器后汇入，不参与滤器内浓度的稀释)*。

**计算逻辑：**
1. **滤器前总钙量** = (前置换液中携带的钙 × 前置换流速) + (患者体内血钙浓度 × 血流速)
2. **滤器前总体积** = 前置换流速 + 血流速 + 抗凝剂流速
3. **混合后理论总钙浓度** = 滤器前总钙量 ÷ 滤器前总体积
4. **枸橼酸根理论总浓度** = 抗凝剂中枸橼酸当量 × 抗凝剂流速 ÷ 滤器前总体积
5. **螯合反应后游离钙估算**：
   - 系统内置非线性螯合系数，经验公式简化为：`估计滤器钙 = 混合后总钙 - (混合后枸橼酸总浓度 × 0.28)`
   - （同时系统设定了极限最低值防止负数异常）。

### F. 特殊定制化液体支持 (Custom Fluids Support)
工具内置常见制式：
- **常规操作**：基础生理盐水等。
- **枸橼酸钠血滤置换液 (5000ml)**：纯枸橼酸置换液配置，系统会自动屏蔽其它微调选项防止不当掺合。
- **磷/碳酸氢钠血滤置换液 (百特, 5000ml)**：自带 1 mmol/L 磷和 22 mmol/L 碳酸氢根，系统解析时自动计入综合离子的累加计算中并输出 HPO₄²⁻ 参数。

## 技术架构与部署 (Technical Stack)
- **技术栈**：纯原生 HTML5, CSS3 结合轻量 Vanilla JavaScript，无外部依赖。
- **样式引擎**：通过 Tailwind CSS 编写高度响应式的 UI (Mobile First 流式网格+弹性布局)。
- **部署**：下载/拷贝 `index.html` 即可在现代主流浏览器直接打开运行，**完全离线可用**，保障临床数据隐私。
